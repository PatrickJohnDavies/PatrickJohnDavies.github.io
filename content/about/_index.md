# About

Hi, I’m Patrick.

I use this site to document my software and electronics projects. Some posts are polished writeups, while others are rough engineering notes from active development.

## What I build

- Firmware for embedded devices
- Desktop applications
- USB-connected hardware
- Data acquisition tools
- Test fixtures and lab utilities

## Why this site exists

I wanted a simple place to keep project notes that are easier to browse than random folders, screenshots, and half-finished text files.
